Para compilar el programa, instalar las dependencias necesarias:
- Entorno virtual:
    - python -m venv venv
    - source venv/bin/activate (Linux/Mac) o venv\Scripts\activate (Windows)
- Instalar dependencias:
    - pip install -r requirements.txt
- Ejecutar el programa:
    - python main.py

- Codigo completo en poisson_final.py: python poisson_final.py
